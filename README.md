# Recruiting Intelligence Frameworks

Hey, I'm Jeff 👋

This repository is a collection of recruiting, sourcing, talent intelligence, and candidate evaluation frameworks I've developed, adapted, or continue to refine throughout my career.

Most recruiting resources focus on tools.

This repository focuses on thinking.

The goal is to better understand:

* How exceptional talent develops
* How technical talent can be identified earlier
* Which signals matter most
* How talent markets evolve
* How recruiters can make better decisions

## Why This Repository Exists

Recruiting is often viewed as a process problem.

I believe many recruiting challenges are actually information problems.

Better information leads to:

* Better hiring decisions
* Better candidate experiences
* Better talent strategies
* Better outcomes

This repository is my attempt to document frameworks that help organize and improve those decisions.
